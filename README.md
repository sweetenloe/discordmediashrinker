# Discord Media Shrinker

### Description:
An efficient and short script that'll <b> convert media files to <= 10mb </b> for Discord not to throw a fit because you didn't want to pay for Nitro.

### Usage:

```python
discordmediashrinker.py <input> [output]
```

<b><i> if the file fails it's because of hard physical limits of what you want to be 10mb </b> </i>
